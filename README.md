# Stack-Overflow-Developer-Survey-Analysis-Python-Pandas
Python (Pandas) data analysis project exploring the Stack Overflow Annual Developer Survey to extract global developer compensation insights and trends.

## 📌 Project Overview
Comprehensive analysis of the Stack Overflow Annual Developer Survey using Python. The project focuses on developer compensation, programming language trends, and global market insights.

## 🛠 Tech Stack
* **Python**: Core language for analysis.
* **Pandas**: Used for data filtering, grouping, and complex aggregations.
* **Data Source**: Stack Overflow Annual Developer Survey (CSV).

## 💡 Key Features & Analysis
* **Data Cleaning**: Handled missing values (NaN) in compensation data and filtered entries for Python developers.
* **Global Statistics**: Aggregated mean and median compensation across 15+ countries to identify market trends.
